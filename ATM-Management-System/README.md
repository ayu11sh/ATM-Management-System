# ATM-Management-System
## A Java swing mini project.

 +The Frames With Corresponding Options Are:
 
 #### 1) LoginFrame
 +	a)login
 +	b)register
 +	c)Exit
 
 #### 2)RegisterFrame
 +	a)accept
 +	b)back
 
 #### 3)SelectionFrame
 +	a)Open Account
 +	b)Deposit
 +	c)Withdraw
 +	d)Transfer
 +	e)Close Account
 +	f)Exit ATM
 
 #### 3)AddAccountFrame
 +	a)savings account
 +	b)checking account
 
 #### 4)DepositFrame
 +	a)Make Deposit
 
 #### 5)WithdrawFrame
 +	a)Make Withdraw
 
 #### 6)TransferFrame
 +	a)Make Transfer
 
 #### 7)InfoFrame
 +	a)return
 
 #### 8)CloseFrame
 +	a)close account
 
 #### 9)AdminFrame
 +	a)Accounts in Alphabetical Order
 +	b)Highest to Lowest Balance
 +	c)Transaction Report
 +	d)Customer Accounts
 +	e)Exit ADMIN
